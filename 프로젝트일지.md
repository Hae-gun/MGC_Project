## 논리적 모델링

1. user.class
   * id (pk) String
   * password String
   * email (이메일 인증) String (unique)
   * location - 도로명 주소 이용. String
   * phone number String
   * logins_State boolean
   
2. place

   * place_id(pk)

   * name
   * position
   * available_time
   * open/close

3. reservation

   * reservation_id(pk)

   * users `map<user,String>` String : roomMaster, none (fk)
   * place (fk)
   * time
